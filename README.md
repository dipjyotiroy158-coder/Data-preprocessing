# Titanic Data Preprocessing 🚢

Project Overview

This project demonstrates the essential data preprocessing steps required before building a machine learning model.
The dataset used contains preprocessed and encoded features.


Steps Performed

* Handling missing values using SimpleImputer
* Splitting dataset into training and test sets
* Feature scaling using StandardScaler


Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

Dataset

* Preprocessed dataset with encoded categorical variables
